# Task TODO CLI

Aplicacion de Tareas por hacer con sus respectivo CRUD.

## Requirements

- Node.js
- npm or yarn

## Install

Clona este repositorio con el siguiente comando:
```
git clone https://github.com/Wolffoner/TODO-cli
```
Luego instala las dependencias

Si usas npm:
```
npm i
```

Si usas yarn:
```
yarn
```
